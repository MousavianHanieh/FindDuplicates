### Find Duplicates

Finds and prints all duplicate numbers in an integer array.  

**Description:**  
This program compares each element in the array with the rest of the elements.  
If a duplicate is found, it is added to a list and printed at the end.

**Example:**  
Input → `[1,6,4,1,17,3,17,4,5,12,2,14,5,2]`  
Output →  
[1, 4, 17, 4, 5, 2]
